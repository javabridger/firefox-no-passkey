# Privacy Policy — NoPasskey

**Last updated: May 2026**

NoPasskey collects no personal data whatsoever.

## What the extension stores

NoPasskey stores only your own preferences — the blocking toggles you set and
the list of sites you have added to your allowlist. This data is stored using
Firefox's built-in extension storage (`browser.storage.sync` when available,
falling back to `browser.storage.local`). It is never transmitted to any
external server.

## What the extension does not do

- It does not collect, transmit, or sell any data.
- It does not track which sites you visit.
- It does not communicate with any server outside your browser.
- It has no analytics, telemetry, or crash reporting.

## Permissions used

| Permission | Why |
|---|---|
| `storage` | Save your blocking preferences and allowlist locally / via Firefox Sync |
| `activeTab` | Read the current tab's URL to show per-site status in the popup |
